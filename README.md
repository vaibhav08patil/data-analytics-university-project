# 📊 Top 1000 Universities Data Analysis

## 📌 Project Description
This project analyzes the Top 1000 Global Universities dataset using Python to understand ranking patterns and performance factors.

## 🔧 Tools Used
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## 📈 Features
- Data Cleaning
- Data Visualization (multiple charts)
- Correlation Analysis
- Linear Regression Model

## 📊 Insights
- USA and UK have the highest number of top universities
- Higher scores improve rankings
- Strong correlation exists between performance indicators

## 📁 Files Included
- data.csv
- notebook.ipynb
